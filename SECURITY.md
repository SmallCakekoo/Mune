# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not create a public issue**.

Instead, send an email directly to:
**Me** - [payan.sary@gmail.com](mailto:payan.sary@gmail.com)
Or send me a message via discord: smallcakeko

### What to include in your report:
- Type of vulnerability
- Steps to reproduce
- Potential impact
- Any exploit code (if possible)

I will try to respond within 48 hours.
