# Contribute to Mune

Thank you for your interest in Mune! Although this is primarily a personal project, I appreciate feedback and suggestions.

## How to Contribute

### Report Bugs
1. Check if the bug already exists in [issues](https://github.com/SmallCakekoo/Mune/issues)
2. Create a new issue with:
- Steps to reproduce the bug
- Expected vs. actual behavior
- Screenshots, if possible

### Suggest Features
1. Describe the feature you would like to see
2. Explain why it would be useful
3. Open to discussing implementations!

### Development
If you want to contribute code:
1. Fork the repository
2. Create a branch for your feature
3. Follow the existing code style
4. Open a Pull Request describing the changes
